# Trabajo de Fin de Máster - Gagik Danielyan Tadevosyan

Se utiliza el código incluido en este repositorio para el Trabajo de Fin de Máster de Ciencias Actuariales y Financieras.

Se trabaja con la base de datos STATS19.

Los datos históricos deben descargarse manualmente desde:

https://www.gov.uk/government/statistical-data-sets/road-safety-open-data

Después es necesario modificar la ruta del archivo CSV en el script. Ahora mismo está como:

csv_hist <- "cambiar/ruta/descargando/antes/dft-road-casualty-statistics-collision-1979-latest-published-year.csv"

Los datos más recientes (2020–2024) se descargan automáticamente mediante el paquete stats19.
